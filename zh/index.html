什么是 Proton 质子链?

Proton质子链通过区块链技术与有效激励机制，为现有网站与应用提供低成本的UID体系解决方案，让所有数字营销领域内各方，都能提升、进化精准数字营销能力与效果。 质子链是一条公链，致力于数据确权和数据协作，先落地于数字营销行业。我们基于用户统一ID系统做底层数据存储和协作，确权广告主的投放数据和用户的行为数据，助力现有的数字营销生态。为广告主，4A广告公司，数据公司，DSP平台及终端用户等提供底层区块链支持和数据服务，为各环节参与者提供价值。

Proton质子链致力于建造全网数据聚合协作公链，旨在打通全网数据，实现安全、高效的跨数据源协同运算，为大数据及人工智能在多个领域的应用提供基础设施支持。 让小到单一的网店大到IoT产业数据都可以通过质子链Proton Inside实现数据彼此之间的聚合与协作，帮助各个行业突破因数据有效样本不足、数据可识别性低的发展瓶颈，释放更多商业价值。

行业挑战

数字营销生态里的参与者之间存在很多不信任的环节

投放效果无法准确衡量

一大部分广告投放不能被很完整的端到端追踪与衡量，用户的操作轨迹很多时候并不是按照预设的步骤进行。导致流量主无法最大化利润，广告主无法衡量投放回报。

投放效果无法预测

在目前的互联网环境下，产品之间大多相互孤立，建造了一个个“信息孤岛”。同一个用户在不同App上只存留少量行为与信息，而大量的脱敏行为数据由于商业原因并不会公开或者与其他App进行置换。导致广告主并不能通过完整的用户画像进行精准投放。

多方利益难以调和

广告主希望投放开销更合理，媒介方希望多赚取渠道费用， 而用户的体验则在前两者的博弈中受到影响。但却没有一个很好的衡量体制。

解决方案

质子链引入三个核心基础概念来解决以上困局。

统一用户识别( Unified Identity )

* 每一个用户的全网统一ID
* 基于UID用户可以做到全网产品之间的用户画像互补

用户行为轨迹做到完整追踪与记录，从而打破“信息孤岛”效应。

统一用户画像( Unified Profile )

* 用户统一的画像与数据记录。
* Includes:
    * 用户基本数据，并基于密码学加密使得数据更具备安全性
    * 用户完整的行为数据
    * 标签化数据，基于全样本数据分析和挖掘

统一画像使流程可追踪溯源，更容易区分用户行为真实与否。

统一注意力( Unified Attention )

* UA是用来衡量一个特定用户（基于UP）与内容本身的综合量化指标
UA基于当前用户的UP和当前行为综合衡量出，用户产生的价值

从本质上解决了用户，流量主和广告主的利益分配问题。
